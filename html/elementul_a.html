Elementul A din HTML5

Elementul <a> din HTML5 este utilizat pentru a crea legături hipertext (hyperlinks), care permit navigarea între diferite pagini web, secțiuni ale aceleiași pagini sau resurse externe. Este unul dintre cele mai importante și utilizate elemente din HTML.

Structura generală
Cod HTML5 

<a href="URL">Text sau conținut vizibil</a>
Atribute principale
  1. href (Hypertext Reference):
    • Specifică URL-ul destinației.
    • Exemple:
      • Link extern: <a href="https://www.example.com">Exemplu</a>
      • Link intern: <a href="/pagina-interna.html">Pagina internă</a>
      • Ancoră: <a href="#sectiune">Secțiune</a> (navigare într-o anumită secțiune din aceeași pagină)
      • E-mail: <a href="mailto:exemplu@email.com">Trimite e-mail</a>
  2. target:
    • Controlează modul în care link-ul se deschide.
    • Valori comune:
      • _self (implicit): Deschide în aceeași fereastră.
      • _blank: Deschide într-o fereastră sau un tab nou.
      • _parent: Deschide în fereastra părinte.
      • _top: Deschide în fereastra principală.
  3. rel (Relationship):
    • Definește relația dintre pagina curentă și link.
    • Valori comune:
      • noopener: Previne accesul paginii deschise la contextul window.opener.
      • noreferrer: Nu trimite informații despre pagina de origine.
      • nofollow: Indică motoarelor de căutare să nu urmeze linkul.
  4. type:
    • Specifică tipul MIME al resursei la care face legătura.
    • Exemplu: <a href="document.pdf" type="application/pdf">Descarcă PDF</a>
  5. download:
    • Permite descărcarea fișierului specificat, în loc să îl deschidă.
    • Exemplu: <a href="document.pdf" download>Descarcă PDF</a>
  6. title:
    • Adaugă un text descriptiv care apare ca tooltip.
    • Exemplu: <a href="https://www.example.com" title="Mergi la exemplu">Exemplu</a>
  7. hreflang:
    • Specifică limba resursei linkate.
    • Exemplu: <a href="https://es.example.com" hreflang="es">Exemplu în spaniolă</a>
  8. accesskey:
    • Permite navigarea rapidă printr-o tastă de acces.
    • Exemplu: <a href="/pagina" accesskey="p">Pagina</a> (Alt + P în majoritatea browserelor).
  9. id și class
    • Folosite pentru identificare unică și aplicarea stilurilor CSS sau manipularea prin JavaScript.
Conținut permis

În interiorul unui tag <a> se pot adăuga diverse elemente, cum ar fi:

  • Text simplu: <a href="https://example.com">Click aici</a>
  • Text simplu: <a href="https://example.com"><img src="imagine.jpg" alt="Exemplu"></a>
  • Alte elemente HTML (span, div, etc.).
Exemple complexe
  1. Link cu deschidere într-un tab nou și atribut rel:
    Cod HTML5 
            
        <a href="https://example.com" target="_blank" rel="noopener noreferrer">Exemplu</a>
  2. Descărcare fișier cu nume personalizat:
    Cod HTML5 
            
        <a href="document.pdf" download="Ghid.pdf">Descarcă ghidul</a>
  3. Navigare către o secțiune specifică:
    Cod HTML5 
            
        <a href="#sectiunea1">Du-te la secțiunea 1</a>
        <h2 id="sectiunea1">Secțiunea 1</h2>
  4. Link pentru trimitere de e-mail:
    Cod HTML5 
            
        <a href="mailto:exemplu@email.com?subject=Salut&body=Mesaj">Trimite un e-mail</a>
  5. Link stilizat cu CSS:
    Cod HTML5 
            
        <a href="/pagina-interna.html" class="buton">Mergi la pagina internă</a>
        <style>
            .buton {
                background-color: #4CAF50;
                color: white;
                padding: 10px 20px;
                text-decoration: none;
                border-radius: 5px;
            }
            .buton:hover {
                background-color: #45a049;
            }
        </style>

Elementul <a> este esențial în crearea de site-uri web, oferind flexibilitate pentru navigare, descărcare și interacțiuni diverse.
