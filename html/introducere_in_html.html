Introducere în HTML5

HTML5 (HyperText Markup Language, versiunea 5) este cea mai recentă versiune a limbajului standard utilizat pentru structurarea și prezentarea conținutului pe web. Este fundamentul oricărei pagini web și reprezintă scheletul pe care se bazează toate tehnologiile web moderne.

De ce este necesar HTML5?

HTML5 aduce îmbunătățiri semnificative față de versiunile anterioare, răspunzând nevoilor actuale ale web-ului modern:

  • Structură semantică îmbunătățită: Include elemente precum <header>, <footer>, <article>, și <section> care fac structura paginii mai clară și mai ușor de înțeles atât pentru dezvoltatori, cât și pentru motoarele de căutare.
  • Suport multimedia nativ: Permite includerea directă a fișierelor audio și video prin elementele <audio> și <video>, eliminând necesitatea pluginurilor externe.
  • Funcționalități interactive: HTML5 lucrează perfect cu CSS și JavaScript pentru a crea aplicații web interactive.
  • Suport pentru aplicații web mobile: Include funcționalități optimizate pentru dispozitive mobile, cum ar fi geolocalizarea, stocarea locală și suportul pentru ecrane tactile.
  • Performanță îmbunătățită: Codul HTML5 este mai curat, mai eficient și mai ușor de interpretat de către browsere.

Cum să creezi un fișier HTML5?

  • Structura de bază a unui fișier HTML5: Fiecare fișier HTML începe cu un doctype care indică utilizarea HTML5. Structura de bază a unui fișier HTML5 este următoarea:

Exemplu de Cod HTML5

Cod HTML5 
        
<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplu HTML5</title>
</head>
<body>
    <header>
        <h1>Bun venit în lumea HTML5!</h1>
    </header>
    <section>
        <article>
            <h2>Ce este HTML5?</h2>
            <p>HTML5 este limbajul standard pentru crearea paginilor web moderne.</p>
        </article>
    </section>
    <footer>
        <p>© 2024 Daniel Savescu</p>
    </footer>
</body>
</html>
  • Explicația structurii:
    • <!DOCTYPE html>: Indică faptul că se folosește HTML5.
    • <html lang="ro">: Marchează începutul documentului HTML și specifică limba (română).
    • <head>: Conține metadatele documentului (titlu, setul de caractere, viewport pentru responsive).
    • <body>: Conține conținutul vizibil al paginii.
    • <header>, <section>, <article>, <footer>: Conține conținutul vizibil al paginii.

Cum să creezi un fișier HTML5?

  1. Salvează fișierul cu extensia .html, de exemplu, pagina.html
  2. Deschide fișierul într-un browser (de exemplu, Chrome, Firefox sau Edge).
  3. Vei vedea pagina structurată conform codului scris.

Aceasta este baza pentru orice proiect web. De aici poți începe să adaugi stiluri (CSS) și funcționalități interactive (JavaScript) pentru a crea aplicații web complexe!
