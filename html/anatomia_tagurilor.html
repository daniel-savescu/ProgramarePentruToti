Anatomia tag-urilor HTML

Tag-urile HTML reprezintă elementele de bază ale structurii unui document HTML. Fiecare tag are un rol specific în organizarea și afișarea conținutului unei pagini web. Să detaliem anatomia unui tag HTML tipic:

  1. Tag-ul de început (Opening Tag)

    Aceasta este partea care definește începutul unui element HTML. Este formată dintr-un nume de tag înconjurat de paranteze unghiulare (< >).

    Sintaxă:

    Cod HTML5 
    
<nume_tag atribut="valoare"></nume_tag>

    Exemple:

    • <p> - definește începutul unui paragraf.
    • <img src="imagine.jpg" alt="Descriere imagine"> - definește o imagine cu atribute.
  2. Atributele

    Atributele sunt utilizate pentru a adăuga informații suplimentare unui element HTML. Ele apar în interiorul tag-ului de început și sunt formate dintr-o pereche nume=valoare

    Caracteristici:

    • Atributele pot fi opționale.
    • Sunt scrise sub formă de perechi: atribut="valoare".
    • Pot influența comportamentul și stilul elementului.

    Exemple de atribute comune:

    • id: identificator unic pentru element.
    • class: specifică una sau mai multe clase CSS.
    • style: aplică stiluri inline.
    • src: definește sursa unui fișier extern, de exemplu pentru imagini sau scripturi.
    • alt: descrie o imagine pentru accesibilitate.

    Exemplu:

    Cod HTML5 
    
<img src="example.jpg" alt="O descriere utilă" class="imagine">
  3. Conținutul (Inner Content)

    Conținutul reprezintă textul sau alte elemente HTML care sunt incluse între tag-ul de început și tag-ul de închidere.

    Exemple:

    • Pentru un paragraf:
      Cod HTML5 
          
    <p>Aceasta este o propoziție într-un paragraf.</p>
    • Pentru un element <div> cu alte elemente HTML în interior:
      Cod HTML5 
              
        <div>
            <h1>Titlu</h1>
            <p>Text în interiorul unui div.</p>
        </div>
  4. Tag-ul de închidere (Closing Tag)

    Aceasta marchează sfârșitul unui element HTML. Este similar cu tag-ul de început, dar include o bară oblică (/) înainte de numele tag-ului.

    Sintaxă:

    Cod HTML5 
    
</nume_tag>

    Exemple:

    • </p> pentru paragrafe.
    • </div> pentru containere.
  5. Elemente care nu necesită tag-uri de închidere

    Unele elemente HTML sunt auto-închise și nu necesită un tag de închidere explicit. Acestea sunt cunoscute sub denumirea de elemente self-closing. Ele includ o bară oblică la sfârșitul tag-ului de început (opțional în HTML5).

    Exemple:

    • <img src="image.jpg" alt="Descriere"> - afișează o imagine.
    • <br> - introduce o pauză de linie.

    Structura completă a unui element HTML:

    Un element HTML tipic include toate componentele de mai sus:

    Cod HTML5 
    
<tag atribut1="valoare1" atribut2="valoare2">Conținut</tag>

    Exemplu concret:

    Cod HTML5 
    
<a href="https://www.google.com" target="_blank" class="link-google">Vizitează site-ul</a>
    • <a>: Tag-ul de început, definește un link.
    • href="https://www.google.com": Specifică URL-ul.
    • target="_blank": Deschide link-ul într-o filă nouă.
    • class="link-google": Definește o clasă CSS.
    • Vizitează site-ul: Conținutul afișat ca text al link-ului.
    • </a>: Tag-ul de închidere.
  6. Contextul elementelor în structură

    Tag-urile HTML pot fi:

    • Blocuri (Block-level elements): Ocupă întreaga lățime a containerului (ex. <div>, <p>, <h1>).
    • Inline elements: Ocupă doar cât este necesar (ex. <a>, <span>).

Aceasta este anatomia unui tag HTML explicată în detaliu, incluzând toate componentele și exemple practice.
