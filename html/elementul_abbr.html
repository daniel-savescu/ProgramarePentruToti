Elementul ABBR din HTML5

Elementul <abbr> din HTML5 este folosit pentru a indica o abreviere sau un acronim. Este util pentru a oferi informații suplimentare despre o abreviere, cum ar fi semnificația completă, prin intermediul atributului title. Când utilizatorii trec cu cursorul peste abrevierea marcată cu <abbr>, majoritatea browserelor afișează valoarea atributului title ca un tooltip.

Sintaxa
Cod HTML5 

<abbr title="Hypertext Markup Language">HTML</abbr>
Explicație:
  1. Abreviere simplă cu tooltip:
    Cod HTML5 
        
    Învață să scrii cod în <abbr title="Hypertext Markup Language">HTML</abbr>.

    Aici, utilizatorii vor vedea textul „Hypertext Markup Language” atunci când trec cu mouse-ul peste „HTML”.

  2. Acronim pentru organizație:
    Cod HTML5 
        
    Suntem membri ai <abbr title="World Health Organization">WHO</abbr>.
  3. Fără atribut title: Dacă <abbr> este folosit fără atributul title, este doar un indicator semantic, fără funcționalitate adițională:
    Cod HTML5 
        
    <abbr>NASA</abbr> a lansat o nouă misiune.
Utilizări practice:
  • Crește accesibilitatea și claritatea conținutului.
  • Ajută utilizatorii care nu cunosc semnificația unei abrevieri.
  • Este util pentru optimizarea SEO, oferind motoarelor de căutare mai multe detalii despre conținut.
Stilizare:

Fiind un element inline, poate fi stilizat cu CSS:

Cod CSS3 

abbr{
    text-decoration: dotted underline;
    cursor: help;
}

Acest element îmbunătățește atât semantica, cât și accesibilitatea paginilor web.
