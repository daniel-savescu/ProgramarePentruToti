Elementul ADDRESS din HTML5

Elementul <address> din HTML5 este utilizat pentru a oferi informații de contact legate de autorul sau proprietarul unui document, articole, pagini web sau secțiuni ale acestora. Este un element semantic care ajută la îmbunătățirea structurii și accesibilității conținutului.

Caracteristici principale:
  • Semnificație semantică: Indică informații despre contact, cum ar fi: nume, adrese de e-mail, adrese poștale, numere de telefon etc.
  • Context de utilizare: Este de obicei folosit pentru a marca detalii de contact asociate întregului document sau unei secțiuni specifice.
  • Stilizare implicită: Browserele aplică de obicei stiluri implicite, cum ar fi text cursiv (italic), dar acest lucru poate fi modificat prin CSS.
Regulile de utilizare:
  1. Plasare:
    • Poate fi folosit în orice secțiune HTML.
    • Este recomandat să fie utilizat doar pentru informații legate de contact.
  2. Continutul acceptat:
    • Poate conține text liber, hyperlink-uri, alte elemente inline.
    • Nu trebuie folosit pentru informații care nu sunt de contact (de exemplu, locația geografică a unui eveniment).
Exemple:

Informații de contact pentru întreg documentul:

Cod HTML5 

<footer>
    <address>
        Contactați-ne la: <a href="mailto:support@example.com">support@example.com</a><br>
        Telefon: +123 456 7890<br>
        Adresa: Str. Exemplu, Nr. 10, Oraș, Țară
        </address>
</footer>

Informații pentru un articol:

Cod HTML5 

<article>
    <h2>Articol despre HTML5</h2>
        <p>Conținutul articolului...</p>
            <address>
                Scris de: <a href="mailto:autor@example.com">autor@example.com</a>
            </address>
</article>
Note:
  • Nu trebuie utilizat pentru a furniza adrese geografice care nu sunt asociate cu informațiile de contact.
  • Este un element semantic care îmbunătățește accesibilitatea, permițând cititoarelor de ecran să identifice ușor informațiile de contact
